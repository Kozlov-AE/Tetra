1. Скачиваем java с офф сайта
  2. Распаковываем в папку, в которую будет установлена Java
  3. Установка в систему (создаются ссылки где нужно)

    4. sudo update-alternatives --install "/usr/bin/java" "java" "Папка, куда распаковл архив/bin/java" 1500

    sudo update-alternatives --install "/usr/bin/javaws" "javaws" "Папка, куда распаковл архив/bin/javaws" 1500

  4. Редактируем environment

    5. sudo nano /etc/environment

  5. Добовляем в /etc/environment следующее:

    6. JAVA_HOME="/usr/lib/jvm/foldername"

  6. Для применения изменении без перезагрузки

    7. source /etc/environment

  7. Проверка версии java

java -version
