Для создания ярлыка для запуска из панели юнити следует сделать такие действия:

1. Создать файл с расширением *.desktop, например run.desktop.


2. Открыть его на редактирование и заполнить следующим:

[Desktop Entry]

Name=Имя ярлыка

Comment=Запускаем свое приложение в терминале

GenericName=Запускаем свое приложение в терминале

Keywords=Мой скрипт

Exec=/usr/bin/gnome-terminal -e /home/user/script.sh

Terminal=false

Type=Application

Icon=/usr/share/pixmaps/faces/dice.jpg

Categories=

Path=

NoDiplay=false


Здесь:

Name - имя ярлыка - как  он будет называться в папке и при наведении на него на панели юнити.

Comment - комментарий к ярлыку

GenericName - описание ярлыка

Keywords - ключевые слова

Exec - команда для запуска


Для запуска скриптов используется такая запись:


/usr/bin/gnome-terminal -e /home/user/script.sh


Для запуска например удаленного рабочего стола, нужно написать так:

rdesktop 99.99.99.99 -u user


Icon - имя файла с картинкой для изображения ярлыка.

Заполнять удобнее из свойств ярлыка - смотреть ниже.

Остальное можно оставить как есть.


3. Обязательно изменить права получившемуся файлу:

chmod a+x run.desktop


4. Кликнуть на получившемся ярлыке правой клавишей - выбрать свойства, и выбрать для него свою картинку.


5. Перетащить ярлык на панель.
